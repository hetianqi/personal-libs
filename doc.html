概述

本文将目前项目中常用的控件列表列出进行API说明，后续也会持续更新...

由于这些控件都是基于bootstrap进行扩展的，因此在页面中需要先引入bootstrap相关的css和js文件和字体文件，针对IE8，还需要引入html5shiv.min.js和respond.min.js文件。 页面中引入以上文件的代码如下： 

							
<head>
    <meta charset="utf-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width,minimum-scale=1.0,maximum-scale=1.0,user-scalable=no">
	<title>Title</title>
	<link rel="stylesheet" type="text/css" href="./assets/css/bootstrap.min.css">
	<link rel="stylesheet" type="text/css" href="./assets/css/common.css">
	<link rel="stylesheet" type="text/css" href="./assets/css/plugins.css">
	<link rel="stylesheet" type="text/css" href="./assets/css/tlayer.css">
	<!--[if lt IE 9]>
		<script src="./assets/js/html5shiv.min.js"></script>
		<script src="./assets/js/respond.min.js"></script>
	<![endif]-->
	<script type="text/javascript" src="./assets/js/jquery-1.11.3.js"></script>
	<script type="text/javascript" src="./assets/js/bootstrap.min.js"></script>
	<script type="text/javascript" src="./assets/js/jquery.util.js"></script>
	<script type="text/javascript" src="./assets/js/jquery.plugins.js"></script>
	<script type="text/javascript" src="./assets/js/jquery.uploadify.js"></script>
	<script type="text/javascript" src="./assets/js/jquery.tlayer.js"></script>
</head>

其中，common.css是针对bootstrap进行reset以及对一些公用样式进行封装的样式文件；tlayer.css和jquery.tlayer.js是弹出框、提示框和加载框插件；jquery.uploadify.js是上传插件（样式文件在plugins.css中）；plugins.css和jquery.plugins.js是常用的插件集合（包括ajax分页、快速生成表格、下拉框美化、滚动条美化等）；jquery.util.js文件对常用函数（如获取url中的键值对、获取浏览器版本信息等）进行了封装。使用者可以按需引入或全部引入。

下面将对除bootstrap自带以外的控件进行说明，根据bootstrap标准，插件提供两种调用方法，data-api方法和纯javascript的调用方式。

单行文本框

给单行文本框添加.form-control类可以获得边框、圆角等效果，添加data-uitype="validate"属性可以为控件赋予正则表达式验证的功能，表达式可以再data-reg属性中设置。data-reg的值分为两部分，中间以空格分开，符合new RegExp(pattern, attributes)的格式 

							
<!--html-->
<input type="text" class="form-control doc-validate" data-uitype="validate" data-reg="^a$" />
							
							
$('.doc-validate').on('validate.ui.validate', function (e, result) {
	$('#input-validate-result').text('验证结果为：' + result);
});

正则表达式验证的结果会通过触发validate.ui.validate事件告诉用户，在回调函数function(e, result){}的第二个参数中

多行文本框

给多行文本框添加.form-control类可以获得边框、圆角等效果。

筛选功能

下拉选择框提供输入筛选功能，只需要为其添加data-searchbox="true"即可实现。

多选功能

data-mulitiple属性表示下拉框的种类，目前有三种（0， 1， 2），分别代表单选框，多选框1和多选框2，默认为单选框。

参数

详细配置参数如下所示，可以将选项通过data属性或 JavaScript 代码传递。对于 data 属性，需要将参数名称放到data-之后，例如data-searchbox="1"

							
{
    name        : '',                      //作为表单的name
    data        : false,                   //数据
    fields      : [],
    /*[{
        name    : 'text',                  //多列字段的名称
        width   : '100%'                   //多列字段所占宽度
    }]*/
    width       : 0,                       //宽度
    seprator    : ';',                     //多选值分隔符
    disabled    : false,                   //是否禁用
    before      : false,                   //是否在前方插入
    searchbox   : false,                   //是否显示搜索框
    multiple    : false                    //是否多选以及多选框类型
}

方法

setValue(value)

为下拉框设置选中项，接收一个字符串（单选）或数组（多选）参数。

setDisabled(disabled)

设置下拉框的禁用状态，接收一个bool值参数，true表示禁用，false为启用。

destroy()

销毁下拉框，并还原select标签

弹出框

弹出框以对话的形式出现，通过某些事件（如click）触发，具有多种类型和实用的功能。 

							
$('#example-tlayer-msg').click(function () {
	$.msg('这是消息框');
});

$('#example-tlayer-tips').click(function () {
	$.tips('这是提示框');
});

$('#example-tlayer-loading').click(function () {
	$.loading('这是加载框');
});

$('#example-tlayer-alert').click(function () {
	$.alert('这是警告框');
});

$('#example-tlayer-confirm').click(function () {
	$.confirm('这是询问框');
});

$('#example-tlayer-content').click(function () {
	$.content({
		header: '内容框',
		content: {
			html: '这是内容框'
		},
		footer: [{
			text: '保存',
			style: 'primary'
		}, {
			text: '取消'
		}]
	});
});

种类

所有的弹出框都是基于content内容框改造的。

消息框 $.msg(msg, fn)

消息框接收两个参数。第一个参数必选，为字符串，表示消息文本；第二个参数可选，为函数，表示消息框关闭后回调的函数。

提示框 $.tips(msg, type, fn)

提示框接收三个参数。第一个参数必选，为字符串，表示消息文本；第二个参数可选，如果为函数，则表示提示框关闭后回调的函数，此时消息类型type参数值为1；第三个参数如果存在，则为提示框关闭后的毁掉参数，此时第二个参数表示提示消息类型，可选值有0，1，2，3，对应图标 的位置。

加载框 $.loading(msg)

加载框接收一个必选参数，为字符串，表示消息文本。

警告框 $.alert(title, msg, callback)

警告框接收三个参数。第一个参数必选，为字符串，表示警告框标题；第二个参数可选，如果为字符串，则为消息内容，如果为函数，则表示用户点击确定之后的回调函数，此时第一个参数为消息内容；第三个参数可选，为函数，表示用户点击确定之后的回调函数。

询问框 $.confirm(title, msg, callback)

询问框参数与警告框相同。

自动化表格

自动化表格插件将重用的表格功能进行封装，提供固定表头滚动、表头点击排序、单元格编辑、列宽度拖动、树形表格等功能。 

							
<div id="example-table"></div>
							
													
$("#example-table").table({
	url: "https://api.douban.com/v2/movie/top250?start=1",
	crossDomain: true,
	data: function () {
		return {
			keyword: $('#keyword').val()
		};
	},
	indexField: "start",
	sizeField: "count",
	dataField: "subjects",
	height: 400,
	pageSize: 20,
	colOptions: [{
		name: 'ID',
		field: 'id',
		edit: {
			replace: function (rowData) {
				return '<input value="'+rowData.id+'" />';
			}
		},
		sort: {
			sname: 'id',
			sorder: 'asc'
		},
		handler: function (value, data) {
			return '<a href="'+data.alt+'" target="_blank">'+data.id+'</a>';
		}
	}, {
		name: '名称',
		field: 'title'
	}, {
		name: '原始名称',
		field: 'original_title'
	}, {
		name: '年代',
		field: 'year'
	}, {
		name: '类型',
		field: 'genres'
	}, {
		name: '收藏数',
		field: 'collect_count'
	}],
	tableClass: "table-hover"
});

照片查看器

        data: imgArr,

        loadingimg: "./assets/img/loading_32.gif"
    });
});

选项

照片查看器的详细配置参数如下： 
							
{
    data        : [],                               //数据
    /*
    [
        {
           src  : '',                               //图片路径
           alt  : ''                                //图片显示名称，如果省略则使用图片名称
        }
    ]        
     */

}

滚动条美化

为一个出现滚动条容器添加data-uitype="scrollbar"属性可获得滚动条美化的效果。

滚动条美化

							
<div data-uitype="scrollbar" style="width: 400px; height: 400px; overflow: auto; margin: 0 auto; border: 1px solid #ddd;">
	<p style="width: 500px; height: 500px; line-height: 500px; text-align: center;">滚动条美化</p>
</div>

选项

滚动条美化的详细配置参数如下： 
							
{
    scrollbarSize   : 8,                    //滚动条宽度
    container       : $container,           //容器
    onInit          : false,                //初始化完毕回调
    onScroll        : false                 //滚动时回调
}

宽度拖动

为一个容器使用添加data-uitype="resize"属性可动态改变容器的宽度。

将鼠标放到左边框上试试
							
<div data-uitype="resize" style="width: 400px; height: 400px; border: 1px solid #ddd;"></div>

选项

由于宽度拖动一般伴随着回调函数的绑定，因此建议使用javascript调用的方式，$().uiResize(options)详细配置参数如下： 
							
{
    vertical        : true,                         //拖动方向，true为垂直方向
    border          : 1,                            //可拖动区域的宽度
    minWidth        : 0,                            //容器可改变的最小宽度
    minHeight       : 0,                            //容器可改变的最小高度
    maxWidth        : Infinity,                     //容器可改变的最大宽度
    maxHeight       : Infinity,                     //容器可改变的最大高度
    onResizeStart   : function (e) {},              //开始拖动的回调，返回false的不进行拖动
    onResize        : function (e, distance) {},    //拖动中的回调
    onResizeEnd     : function (e) {}               //拖动结束的回调
}
